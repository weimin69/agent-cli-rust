# LEARNING ROADMAP

## Vision

Build a production-quality Agent CLI in Rust.

The project should eventually become a modern AI Agent command-line application with clean architecture, good engineering practices, and extensible design.

Learning always serves this final objective.

---

# Stage 1 — Rust Fundamentals

Goal:

Become comfortable writing small Rust CLI programs.

Topics:

- Cargo
- Modules
- Ownership
- Borrowing
- Struct
- Enum
- Match
- Error Handling
- Result
- Option
- Crates
- Traits
- Generics

Milestone:

Implement a multi-command CLI using Clap.

---

# Learning Principles

The roadmap is a guideline rather than a strict checklist.

Always:

- Learn only what is needed for the current stage.
- Prefer incremental progress.
- Avoid introducing advanced topics too early.
- Connect every new Rust concept to the current project.
- Build real features instead of isolated examples.

---

# Final Goal

By completing this roadmap, the developer should be able to:

- Write idiomatic Rust.
- Build maintainable CLI applications.
- Understand asynchronous Rust.
- Design extensible software architectures.
- Build modern AI Agent systems.
- Release a production-quality Agent CLI written entirely in Rust.
